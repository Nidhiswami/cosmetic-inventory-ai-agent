# Cosmetic Inventory AI Agent

AI-powered cosmetic inventory management system built using Python and Streamlit.

---

## 🚀 Features

- Low stock detection
- Reorder suggestions
- Structured JSON response
- Inventory summary
- Interactive UI

---

## 🛠 Tech Stack

- Python
- Pandas
- Streamlit

---

## 📦 How to Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

---

## 📊 Sample Queries

- Show low stock items
- Show total inventory
- Give inventory summary
- Is stock sufficient?
